# Generated Reports

This page will display reports that you have generated from a template.

## Table Details

| Column        | Description                                                                                                                 |
| ------------- | --------------------------------------------------------------------------------------------------------------------------- |
| Template Name | The name of the template used to generate the report.                                                                       |
| Tenant Filter | The tenant the report was generated for.                                                                                    |
| Generated At  | The relative time since the report was generated.                                                                           |
| Status        | The status of the generated report. The only available status is `Completed`. Any failed reports would note to the logbook. |
| Sections      | The number of sections on the report.                                                                                       |

## Table Actions

<table><thead><tr><th>Action</th><th>Description</th><th data-type="checkbox">Bulk Action Available</th></tr></thead><tbody><tr><td>View Report</td><td>Opens a preview of the PDF generated by the report. </td><td>true</td></tr><tr><td>Delete</td><td>Deletes the selected generated report(s).</td><td>true</td></tr><tr><td>More Info</td><td>Opens the Extended Info flyout.</td><td>false</td></tr></tbody></table>
