## Page Actions

<details>

</details>

<details>

<summary>Browse Catalog</summary>

This will open a flyout where you can browse configured [community-repos](../community-repos/ "mention") for template reports for import.

</details>

## Table Details

| Column       | Description                                              |
| ------------ | -------------------------------------------------------- |
| Name         | The name given to the report                             |
| Sections     | The total number of sections on the report.              |
| Test Count   | The number of CIPP built tests included in the report.   |
| Custom Count | The number of custom built tests included in the report. |
