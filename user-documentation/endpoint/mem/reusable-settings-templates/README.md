## Table Details

| Column       | Description                                                              |
| ------------ | ------------------------------------------------------------------------ |
| Display Name | The display name set on the template                                     |
| Description  | The description entered on the template when created                     |
| Is Synced    | An indicator if the template has been synced to/from a GitHub repository |
