# Apple ADE

This page will list and allow you to manage Apple ADE Profiles. The top has information on the number of ADE Tokens, Expiring Tokens, and ADE Profiles. It will also show the last successful sync.

## Page Actions

<details>

<summary>Sync DEP</summary>

This will trigger CIPP to sync DEP information from Microsoft.

</details>

## Table Details

The properties returned are for the Graph resource type `depOnboardingSetting`. For more information on the properties please see the [Graph documentation](https://learn.microsoft.com/en-us/graph/api/resources/intune-enrollment-deponboardingsetting?view=graph-rest-beta#properties).

<table><thead><tr><th>Action</th><th>Description</th><th data-type="checkbox">Bulk Action Available</th></tr></thead><tbody><tr><td>Delete Profile</td><td>Deletes the selected profile(s)</td><td>true</td></tr><tr><td>More Info</td><td>Opens the Extend Info flyout</td><td>false</td></tr></tbody></table>
