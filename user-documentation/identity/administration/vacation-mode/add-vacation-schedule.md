# Add Vacation Schedule

This wizard allows you to schedule a task to add and remove a user or multiple users from CA exclusions for the specified period of time.&#x20;

{% stepper %}
{% step %}
### Tenant Selection

This will automatically default to the selected tenant from the top menu bar, but you are able to override that at your choice.
{% endstep %}

{% step %}
### User Selection

Select the user(s) you want to apply the vacation schedule to
{% endstep %}

{% step %}
### Vacation Actions

Optionally you can enable CA policy exclusions, grant temporary mailbox and calendar access, and set out of office autoreplies.

{% hint style="info" %}
You must enable at least one of these options to continue.
{% endhint %}
{% endstep %}

{% step %}
### Schedule

Set the scheduled start date, scheduled end date, any post execution actions you want to take (i.e. notifications), and a reference for later review.
{% endstep %}

{% step %}
### Review & Submit

Confirm your submissions and submit if everything looks as you desire.
{% endstep %}
{% endstepper %}
