# ActiveSync Devices

## Table Details

The properties returned are for the Exchange PowerShell command `Get-MobileDevice`. For more information on the command please see the [Microsoft documentation](https://learn.microsoft.com/en-us/powershell/module/exchangepowershell/get-mobiledevice?view=exchange-ps).&#x20;
