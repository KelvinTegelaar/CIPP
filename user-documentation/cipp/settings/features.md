# Features

This page is used to manage the visibility of features that are still in beta or are in the process of being deprecated. To manage the features, click the Actions three dots (alternatively you can select multiple check boxes and use the Bulk Actions dropdown) and select Enable Feature or Disable Feature.&#x20;
