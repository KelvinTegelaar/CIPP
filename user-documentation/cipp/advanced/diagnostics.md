# Diagnostics

This page will allow you to dig in on diagnosing issues with CIPP by viewing Azure Application Insights.

## Query

Here, you will be able to create a query. These queries are written in [Kusto Query Language](https://learn.microsoft.com/en-us/kusto/query/?view=microsoft-fabric). For ease, there are several preset queries already stored for you.

## Diagnostics

This table will display the results of your query. You can select the `More Info` action to view a single result in more detail.
