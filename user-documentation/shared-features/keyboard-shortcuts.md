# Keyboard Shortcuts

The table below outlines the keyboard shortcuts that have been enabled in CIPP.

| Shortcut                  | Windows        | Mac                    |
| ------------------------- | -------------- | ---------------------- |
| Open Search               | Ctrl + K       | Cmd(⌘) + K             |
| Highlight Tenant Selector | Ctrl + Alt + K | Cmd(⌘) + Option(⌥) + K |
