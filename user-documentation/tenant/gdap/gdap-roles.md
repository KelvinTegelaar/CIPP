# GDAP Roles

This page shows all the roles and what security group in the partner tenant, that are available to be mapped to a GDAP relationship via CIPP.

To create new mappings, click the `Map GDAP Roles` button.

## Actions

* Map GDAP Roles
* Delete Mapping
