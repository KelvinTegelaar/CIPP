# Snoozed Alerts

This page will list CIPP alerts that you have snoozed to allow you to easily review and manage them.

## Snoozing Alerts

Included in eligible alerts will be a link to open CIPP to a page to set a snooze on the alert. Select the duration of the snooze (7 days, 14 days, 30 days, or 90 days). The page will display the user you are setting a snooze on. Click `Snooze` to set the selected snooze duration.

## Table Details

The table lists a row of the alerts that have been snoozed, the user the alert is snoozed for, their duration, and who snoozed the alert.

## Table Actions

<table><thead><tr><th>Action</th><th>Description</th><th data-type="checkbox">Bulk Action Available</th></tr></thead><tbody><tr><td>Remove Snooze</td><td>This will remove the snooze for the selected alert(s) allowing normal alerting on the next run.</td><td>true</td></tr></tbody></table>
